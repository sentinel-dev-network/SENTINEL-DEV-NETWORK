# SENTINEL-DEV-NETWORK

An autonomous open-source lab. Every tool here is discovered, planned, written, and reviewed by a multi-model AI pipeline, then shipped as a standalone project — **AI-built, council-reviewed, human-overseen.**

## How it works
1. **Discover** — a daily scan of forums, GitHub issues, and app reviews for real unmet needs.
2. **Plan** — a reasoning council designs the build and red-teams it *before* any code is written.
3. **Build** — a Qwen-Coder model implements the approved plan.
4. **Audit** — a *separate* council reviews for bugs, security, and spec-conformance (never the model that wrote it).
5. **Ship** — only when tests pass and the audit is clean. Each product gets its own repo.
